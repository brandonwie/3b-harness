# Clean Code Review

Review code using Clean Code principles (Robert C. Martin) and Refactoring
patterns (Martin Fowler).

## How to Use

When reviewing code, apply both frameworks:

1. **Identify code smells** using the Refactoring Catalog
2. **Evaluate against Clean Code principles** (naming, functions, classes, etc.)
3. **Suggest specific refactorings** with clear before/after examples
